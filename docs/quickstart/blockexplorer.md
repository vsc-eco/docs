# Blockexplorer

The first block explorer for VSC is already live! You can experience it first hand via the link below. 

[Block explorer 'VSC blocks'](https://vsc.techcoderx.com) by [techcoderx](https://github.com/techcoderx)

Current features include:

- witness pool status
- contract deployments
- VSC block exploration
- VSC transaction exploration
- checking out multisig transaction references
